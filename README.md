# 饮食 Agent

儿童手环上的饮食记录 Agent。不需要孩子手动打卡——系统根据饭点时段和 GPS 位置自动判断"孩子可能在吃饭了"，低打扰地引导孩子拍一张照，AI 识别食物并生成饮食记录，同步给家长。

目标用户：4–10 岁儿童 + 家长

---

## 它能做什么

- **自动感知进食**：综合饭点时段 + GPS 就餐场景打分，达到阈值才触发，不瞎提醒
- **拍照识别**：语音提示孩子拍食物照，AI 识别主食、配菜、零食、餐盘结构
- **置信度分级**：高置信直接进日报，低置信标"待确认"推给家长复核
- **日报 / 周报**：家长端查看三餐记录、营养趋势、待确认项

## 运行方式

**A 段（进食检测 demo）：**
```bash
cd 饮食A段
python serve-https.py
# 浏览器打开提示的本地地址
```

**B 段（完整版）：**
```bash
cd 饮食B段
cp .env.example .env   # 填入 LLM 凭证
npm install
npm run dev
```

## 触发逻辑

| 信号 | 贡献分 |
|---|---|
| 非饭点 | 0 |
| 接近饭点 | 40 |
| 饭点中 | 70 |
| GPS 就餐场景 | +15 |

`P(Eating) = min(100, 饭点分 + GPS分)`，达到 **70** 才触发记录流程。
